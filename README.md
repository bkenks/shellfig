# Shellfig

## Installation
1. Clone repository to local machine
2. Do a `find` for the following tags and configure commands in tagged section per system
	```bash
	configure_post_clone
	```
3. Allow execution of `installer.sh`
	```bash
	chmod +x installer.sh
	```
4.  Execute `installer.sh`
	```bash
	./installer.sh
	```
